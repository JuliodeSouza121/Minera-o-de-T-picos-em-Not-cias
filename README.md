# Minera-o-de-T-picos-em-Not-cias
Este projeto utiliza técnicas modernas de Processamento de Linguagem Natural (PLN) e Inteligência Artificial para organizar automaticamente notícias em grupos temáticos. A solução combina embeddings semânticos, algoritmos de clusterização e um Modelo de Linguagem de Grande Escala (LLM) para identificar tópicos relevantes, entre outras coisas.
